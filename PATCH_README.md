# Registro de Patches

Este arquivo documenta as mudanças aplicadas ao repositório após a versão inicial.  Cada seção corresponde a um patch ou conjunto de patches aplicados.  Para ver todos os patches anteriores, consulte o `CHANGELOG.md`.

## 0.1.0

Estrutura inicial do repositório criada em 23 de março de 2026.  Nenhum patch adicional foi aplicado até o momento.

Futuros patches devem ser documentados aqui com:

* Descrição resumida da mudança.
* Data da aplicação.
* Arquivos modificados.
* Justificativa, se necessário.

Exemplo:

```
## 0.1.1 - 2026-04-01

* Atualizado o schema de mensagens para versão 1.1 para permitir novos campos.
* Modificado `src/io/reader.py` para validar contra o novo schema.
* Adicionados novos exemplos sintéticos compatíveis.
```
