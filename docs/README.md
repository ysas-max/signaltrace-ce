# Documentação

Esta pasta contém documentação técnica adicional, whitepapers e guias de uso para o **signaltrace‑ce**.  A documentação está organizada de forma modular para facilitar a consulta.

## Estrutura

| Documento | Descrição |
| --- | --- |
| `schema_event.md` | Detalha o schema público para eventos observados. |
| `engine_flow.md` | Explica o fluxo interno do motor (normalização, fingerprint, correlação e scoring). |
| `scoring.md` | Descreve o sistema de scoring, níveis e fatores. |
| `api.md` | Especificação da API pública local. |
| `cli.md` | Guia de uso da CLI pública. |
| `contracts.md` | Contrato de saída dos relatórios em JSON. |
| `demo.md` | Guia passo a passo para rodar a demonstração sintética. |

## Contribuições

Contribuidores são incentivados a adicionar novos documentos aqui à medida que o projeto evolui.  Exemplos incluem:

- Estudos de caso usando dados artificiais.
- Relatórios de auditoria de segurança.
- Whitepapers sobre metodologias de correlação e scoring.

Ao adicionar novos documentos, atualize esta lista para facilitar a navegação.
