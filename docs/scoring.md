# Sistema de Scoring

O score é um indicador heurístico entre 0 e 1 e **não é prova**.

## Níveis

- `baixo`: 0.00–0.24
- `moderado`: 0.25–0.49
- `alto`: 0.50–0.74
- `crítico`: 0.75–1.00

## Fatores

- repetição de domínio
- similaridade textual
- marcadores de campanha
- proximidade temporal
- risk flags
